This is an upload and abandon sort of software. It was heavily vibecoded, but it works. I will not be fixing any issues this software may have or develop in the future. 

How to use -
Open the program
  Add PSARC's... - Add individual PSARC files to convert
  Add Folder... - Add an entire folder (and any subfolders) to convert
  Remove - Remove selected addition
  Clear - Clears the entire queue
Once the folder for conversion is selected, select your output destination (Default is same folder as the PSARC's

All you need to do after you have your folder to convert and output location is hit "Convert". It should do the rest on its own.

Have fun, and rock on \m/
